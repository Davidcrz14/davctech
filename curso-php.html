Curso de PHP

Curso 1: PHP Básico a Intermedio

Este curso de PHP está diseñado para principiantes que quieren aprender desde los fundamentos hasta temas de nivel intermedio. El objetivo es proporcionar una base sólida en PHP para el desarrollo web.

Temario Curso PHP Básico a Intermedio

Lección 1: Introducción a PHP
  • ¿Qué es PHP? Historia y evolución.
  • PHP vs otros lenguajes de programación web.
  • Instalación y configuración de PHP (XAMPP, WAMP, MAMP).
  • Primer script PHP: Sintaxis básica.
  • Integración de PHP con HTML.

Práctica: Escribir un script sencillo que muestre texto en una página HTML.

Lección 2: Variables, Tipos de Datos y Operadores
  • Declaración de variables en PHP.
  • Tipos de datos (enteros, cadenas, booleanos, arrays, etc.).
  • Operadores (aritméticos, de asignación, comparación, lógicos).
  • Conversión de tipos.

Práctica: Ejercicios con operaciones básicas y manejo de variables.

Lección 3: Estructuras de Control
  • Sentencias condicionales: if, else, elseif, switch.
  • Bucles: for, while, do-while, foreach.
  • Control de flujo con break y continue.

Práctica: Crear un programa que imprima la tabla de multiplicar de un número usando bucles.

Lección 4: Funciones en PHP
  • Definición y llamada de funciones.
  • Funciones con parámetros y valores por defecto.
  • Funciones con retorno de valores.
  • Alcance de variables (global, local, static).

Práctica: Crear funciones para calcular el área de diferentes figuras geométricas.

Lección 5: Arrays en PHP
  • Creación de arrays: indexados y asociativos.
  • Recorrido de arrays.
  • Arrays multidimensionales.
  • Funciones útiles para arrays: array_push, array_pop, array_merge, count, etc.

Práctica: Crear un sistema de calificaciones almacenadas en un array asociativo y calcular el promedio.

Lección 6: Manejo de Formularios en PHP
  • Envío de datos con GET y POST.
  • Acceso a datos de formularios desde PHP.
  • Validación básica de datos (campo requerido, formato).
  • Prevención de inyecciones de código.

Práctica: Crear un formulario de contacto que envíe datos al servidor y los muestre.

Lección 7: Manejo de Sesiones y Cookies
  • ¿Qué son las sesiones y cookies?
  • Manejo de sesiones en PHP: session_start(), $_SESSION.
  • Creación, lectura y eliminación de cookies: setcookie(), $_COOKIE.
  • Comparación entre sesiones y cookies.

Práctica: Crear un sistema de login simple usando sesiones.

Lección 8: Manejo de Archivos
  • Apertura y cierre de archivos: fopen(), fclose().
  • Lectura y escritura en archivos: fread(), fwrite().
  • Funciones de archivos útiles: file_exists(), file_get_contents(), file_put_contents().
  • Subir archivos al servidor.

Práctica: Crear un sistema que permita subir archivos y almacenarlos en una carpeta del servidor.

Lección 9: Programación Orientada a Objetos (POO) en PHP
  • Introducción a POO: clases, objetos, propiedades y métodos.
  • Encapsulamiento: propiedades y métodos public, private, protected.
  • Herencia y polimorfismo.
  • Uso de constructores y destructores.

Práctica: Crear una clase Vehículo con propiedades y métodos, y extenderla para crear clases derivadas como Auto y Moto.

Lección 10: Manejo de Bases de Datos con MySQL
  • Conectar PHP con MySQL usando mysqli y PDO.
  • Ejecución de consultas SQL: SELECT, INSERT, UPDATE, DELETE.
  • Seguridad y consultas preparadas para evitar SQL Injection.
  • Manejo de errores en la conexión.

Práctica: Crear un CRUD (Crear, Leer, Actualizar, Eliminar) de una tabla de usuarios.

Lección 11: Gestión de Excepciones y Errores
  • Tipos de errores en PHP: errores de sintaxis, advertencias, y excepciones.
  • Manejo de excepciones con try, catch y finally.
  • Uso de funciones de manejo de errores: error_reporting(), set_error_handler().

Práctica: Crear un sistema que maneje errores y excepciones al interactuar con una base de datos.

Lección 12: Buenas Prácticas y Optimización en PHP
  • Estructuración de código y modularización.
  • Mejores prácticas para evitar vulnerabilidades comunes.
  • Optimización de rendimiento: caching, compresión, y uso adecuado de recursos.
  • Uso de herramientas de debugging y profiling en PHP.

Práctica: Refactorización de un proyecto PHP usando buenas prácticas y optimización.

Curso 2: PHP Intermedio/Avanzado Integrado con Laravel

En este curso, los estudiantes aprenderán PHP a nivel avanzado, junto con el framework Laravel. Se enfocará en el desarrollo de aplicaciones modernas con Laravel, aprovechando todas las características avanzadas de este framework.

Temario Curso PHP Avanzado con Laravel

Lección 1: Repaso de PHP Intermedio y Avanzado
  • POO avanzada: namespaces, traits, interfaces, y abstractas.
  • Manipulación avanzada de bases de datos con PDO.
  • Patrones de diseño en PHP: Singleton, Factory, MVC.
  • Principios SOLID y DRY en desarrollo PHP.

Práctica: Ejercicios aplicando patrones de diseño y principios SOLID en proyectos PHP.

Lección 2: Introducción a Laravel
  • ¿Qué es Laravel? Historia y arquitectura.
  • Instalación de Laravel usando Composer.
  • Estructura de un proyecto Laravel.
  • Ciclo de vida de una petición HTTP en Laravel.

Práctica: Crear un proyecto básico de Laravel y entender su estructura.

Lección 3: Rutas y Controladores en Laravel
  • Definición de rutas básicas y avanzadas.
  • Controladores en Laravel: generación y manejo.
  • Envío y validación de parámetros en rutas.
  • Controladores RESTful en Laravel.

Práctica: Crear rutas y controladores para una pequeña API de gestión de artículos.

Lección 4: Vistas y Blade Template Engine
  • Introducción a Blade: el motor de plantillas de Laravel.
  • Uso de plantillas y layouts en Blade.
  • Pasar datos desde controladores a vistas.
  • Directivas y estructuras de control en Blade.

Práctica: Crear una vista que muestre una lista de productos con un diseño modular usando Blade.

Lección 5: Bases de Datos y Eloquent ORM
  • Conexión a bases de datos y configuración de .env.
  • Introducción a Eloquent ORM.
  • Definir modelos y relaciones en Eloquent.
  • Uso de migraciones para crear y modificar tablas.
  • Validación de datos con reglas de Eloquent.

Práctica: Crear un sistema de gestión de usuarios usando Eloquent y migraciones.

Lección 6: Formularios, Validación y Autenticación
  • Creación de formularios con Blade.
  • Validación de formularios y manejo de errores.
  • Sistema de autenticación integrado en Laravel.
  • Personalización del registro y login de usuarios.

Práctica: Implementar un sistema de autenticación de usuarios con Laravel Auth.

Lección 7: Middleware y Seguridad
  • Uso de middleware para controlar el acceso a rutas.
  • Middleware integrado y creación de middleware personalizado.
  • Seguridad en Laravel: CSRF, XSS, y protección contra inyecciones SQL.

Práctica: Implementar middleware para gestionar permisos de usuarios en diferentes rutas.

Lección 8: Envío de Emails y Notificaciones
  • Configuración del sistema de correo en Laravel.
  • Envío de correos electrónicos con Mailables.
  • Notificaciones en Laravel (email, base de datos, SMS).

Práctica: Enviar un email de bienvenida cuando un usuario se registre en la aplicación.

Lección 9: API RESTful con Laravel
  • Creación de una API RESTful.
  • Serialización de datos con Eloquent y respuestas JSON.
  • Uso de controladores API y rutas protegidas.
  • Autenticación con Passport o Sanctum.

Práctica: Crear una API RESTful para gestionar productos y autenticar usuarios.

Lección 10: Testing en Laravel
  • Introducción a PHPUnit para pruebas automatizadas.
  • Pruebas de unidades y pruebas funcionales en Laravel.
  • Creación de pruebas para controladores, modelos y vistas.
  • Pruebas de integración y uso de factories.

Práctica: Escribir pruebas unitarias y funcionales para una pequeña aplicación en Laravel.

Lección 11: Despliegue de Aplicaciones Laravel
  • Configuración para entornos de producción.
  • Despliegue en servidores compartidos y VPS.
  • Uso de Forge o Vapor para despliegue automatizado.
  • Configuración de servidores Nginx o Apache para Laravel.

Práctica: Desplegar una aplicación Laravel en un servidor de pruebas.

Lección 12: Optimización y Buenas Prácticas en Laravel
  • Mejores prácticas en estructura de proyectos Laravel.
  • Optimización de rendimiento: caching, optimización de consultas y compresión de recursos.
  • Herramientas de debugging y profiling en Laravel.
  • Seguridad y monitoreo de aplicaciones en producción.

Práctica: Optimizar una aplicación Laravel y aplicar técnicas de caché y debugging.
