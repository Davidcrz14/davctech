Curso de PHP

¿Qué es PHP?

PHP (Hypertext Preprocessor) es un lenguaje de programación de código abierto, utilizado principalmente para el desarrollo web del lado del servidor. PHP se ejecuta en el servidor, lo que significa que genera HTML u otros tipos de datos (como JSON o XML) que luego se envían al navegador del usuario.

Característica Descripción
Código abierto y gratuito PHP es completamente libre para usar y distribuir.
Fácil de aprender Comparado con otros lenguajes de programación, PHP es fácil de aprender y utilizar para principiantes.
Portabilidad PHP se puede ejecutar en la mayoría de los servidores y sistemas operativos (Windows, Linux, macOS).
Gran soporte comunitario Hay una gran cantidad de recursos disponibles debido a su popularidad.

Historia y evolución de PHP

PHP fue creado en 1994 por Rasmus Lerdorf, inicialmente como un conjunto de scripts CGI (Common Gateway Interface) para rastrear las visitas a su sitio web. Al principio, se llamó "Personal Home Page" (PHP), pero con el tiempo evolucionó a lo que hoy conocemos como Hypertext Preprocessor.

  • PHP 3: Introdujo un motor de scripting que revolucionó su uso.
  • PHP 4: Añadió mejoras de rendimiento significativas.
  • PHP 5: Introdujo características orientadas a objetos.
  • PHP 7: Mejoró drásticamente el rendimiento, reducción del consumo de memoria y nuevas características modernas.

Instalación y configuración de PHP

Para empezar a desarrollar en PHP, necesitamos un servidor web y un entorno donde podamos ejecutar nuestros scripts. Las opciones más comunes son:

  1. XAMPP (Windows, Linux, macOS): Un paquete que incluye Apache (servidor web), MySQL (base de datos), PHP y Perl.
  2. WAMP (Windows): Similar a XAMPP, pero solo para Windows. Incluye Apache, MySQL y PHP.
  3. MAMP (macOS): Paquete de servidor local para macOS, que incluye Apache, MySQL y PHP.

Instalación de XAMPP en Windows:

  1. Descarga XAMPP desde el sitio oficial https://www.apachefriends.org/es/index.html.
  2. Ejecuta el instalador y sigue las instrucciones.
  3. Una vez instalado, abre el Panel de Control de XAMPP y arranca los módulos de Apache y MySQL.
  4. Para probar si PHP está funcionando, crea un archivo llamado index.php en la carpeta htdocs dentro del directorio de XAMPP (por ejemplo, C:\xampp\htdocs\).
  5. Escribe este código dentro de index.php:
index.php 
<?php
    echo "¡Hola, mundo desde PHP!";
?>
  1. Abre tu navegador web y ve a la URL http://localhost/index.php. Si todo está configurado correctamente, verás el mensaje "¡Hola, mundo desde PHP!".

Primer script PHP: Sintaxis básica

PHP utiliza la extensión .php para los archivos. La sintaxis básica de PHP se integra directamente en el HTML usando la etiqueta <?php ... ?>. El código dentro de esta etiqueta se ejecuta en el servidor.

ejemplo.php 
<?php
    // Este es un comentario en PHP
    echo "¡Hola, mundo!";
?>

Explicación del código:

  • <?php ... ?>: Indica el inicio y fin del código PHP dentro de un archivo HTML o PHP.
  • echo: Es un comando en PHP que se utiliza para mostrar texto en la página web.
  • "¡Hola, mundo!": Es una cadena de texto que se muestra cuando se ejecuta el script.

Notas importantes:

  • Los comentarios en PHP pueden ser de una línea (//) o de múltiples líneas (/* ... */).
  • Las instrucciones en PHP suelen terminar con un punto y coma (;).

Integración de PHP con HTML

PHP puede integrarse directamente en un archivo HTML. Esto es lo que hace que PHP sea muy flexible para crear páginas dinámicas.

ejemplo-integrado.php 
<!DOCTYPE html>
<html>
<head>
    <title>Página de prueba en PHP</title>
</head>
<body>
    <h1>Bienvenido a mi página web</h1>
    <p>Este es un párrafo estático en HTML.</p>

    <?php
        echo "<p>Este es un párrafo generado con PHP.</p>";
    ?>
</body>
</html>

Declaración de Variables en PHP

Las variables son espacios en memoria donde almacenamos datos. En PHP, las variables se declaran con el símbolo $. No es necesario declarar el tipo de dato, ya que PHP es de tipado dinámico. Algunas reglas importantes:

  • Deben comenzar con el símbolo $
  • El primer carácter debe ser una letra o guion bajo
  • Son sensibles a mayúsculas y minúsculas
  • No pueden contener espacios, usa guion bajo _ o camelCase
variables.php 
<?php
    $nombre = "Juan";        // String
    $edad = 25;              // Integer
    $precio = 19.99;         // Float
    $es_activo = true;       // Boolean
?>

Tipos de Datos en PHP

PHP soporta varios tipos de datos que puedes utilizar en tus programas. Los más comunes son:

Tipo Descripción Ejemplo
Integer Números enteros $edad = 25;
Float Números decimales $precio = 19.99;
String Cadenas de texto $nombre = "Juan";
Boolean Verdadero o falso $activo = true;
Array Colección de valores $colores = array("rojo", "azul");
NULL Valor especial que representa la ausencia de valor $sin_valor = null;

Manejo de Arrays

PHP tiene dos tipos principales de arrays:

  1. Array indexado: Cada elemento tiene un índice numérico que empieza en 0. Ejemplo: 
    $colores = array("rojo", "azul", "verde");
  2. Array asociativo: Cada elemento tiene una clave (key) en lugar de un índice numérico. Ejemplo: 
    $persona = array("nombre" => "Juan", "edad" => 30, "ciudad" => "Madrid");

Accediendo a los elementos de un array:

<?php
    echo $colores[1]; // Imprime "azul"
    echo $persona['nombre']; // Imprime "Juan"
?>

Operadores en PHP

PHP incluye varios operadores que permiten manipular y comparar valores:

Operadores Aritméticos

Se utilizan para realizar cálculos matemáticos.

Operador Descripción Ejemplo
+ Suma $a + $b
- Resta $a - $b
* Multiplicación $a * $b
/ División $a / $b
% Módulo (resto) $a % $b 
<?php
    $a = 10;
    $b = 3;
    $suma = $a + $b;  // Resultado: 13
    $resta = $a - $b; // Resultado: 7
    $multiplicacion = $a * $b; // Resultado: 30
    $division = $a / $b; // Resultado: 3.333...
    $modulo = $a % $b; // Resultado: 1
?>

Operadores de Asignación

Sirven para asignar valores a las variables.

Operador Descripción Ejemplo
= Asignación simple $x = $y
+= Suma y asigna $x += $y (equivale a $x = $x + $y)
-= Resta y asigna $x -= $y
*= Multiplica y asigna $x *= $y
/= Divide y asigna $x /= $y 
<?php
    $x = 5;
    $x += 2; // $x ahora es 7
    $x *= 3; // $x ahora es 21
?>

Operadores de Comparación

Se utilizan para comparar valores. El resultado siempre será true o false.

Operador Descripción Ejemplo
== Igual a $a == $b
!= Diferente de $a != $b
> Mayor que $a > $b
< Menor que $a < $b
>= Mayor o igual que $a >= $b
<= Menor o igual que $a <= $b 
<?php
    $a = 10;
    $b = 5;
    $resultado = ($a == $b); // false
    $resultado = ($a != $b); // true
    $resultado = ($a > $b); // true
    $resultado = ($a < $b); // false
    $resultado = ($a >= $b); // true
    $resultado = ($a <= $b); // false
?>

Operadores Lógicos

Se usan para combinar múltiples condiciones lógicas.

$condicion1 = true;
$condicion2 = false;

$resultado = $condicion1 && $condicion2; // AND: false
$resultado = $condicion1 || $condicion2; // OR: true
$resultado = !$condicion1;               // NOT: false

Conversión de Tipos en PHP

PHP puede convertir tipos de datos automáticamente cuando sea necesario (conversión implícita), pero también puedes hacerlo de forma manual:

$num_str = "10";    // Cadena
$numero = $num_str + 5; // PHP convierte "10" a número automáticamente
echo $numero;       // Resultado: 15

// Conversión explícita
$num_str = "3.14";
$flotante = (float)$num_str;  // Convertimos a flotante
echo $flotante;  // Resultado: 3.14

Sentencias Condicionales

Las sentencias condicionales nos permiten ejecutar diferentes bloques de código según se cumplan o no ciertas condiciones.

La sentencia if

ejemplo_if.php 
<?php
$edad = 20;

if ($edad >= 18) {
    echo "Eres mayor de edad";
}
?>

La sentencia if-else

ejemplo_if_else.php 
<?php
$edad = 16;

if ($edad >= 18) {
    echo "Eres mayor de edad";
} else {
    echo "Eres menor de edad";
}
?>

La sentencia elseif

ejemplo_elseif.php 
<?php
$nota = 85;

if ($nota >= 90) {
    echo "Excelente";
} elseif ($nota >= 70) {
    echo "Aprobado";
} else {
    echo "Reprobado";
}
?>

La sentencia switch

ejemplo_switch.php 
<?php
$dia = "lunes";

switch ($dia) {
    case "lunes":
        echo "Es lunes";
        break;
    case "martes":
        echo "Es martes";
        break;
    default:
        echo "Es otro día";
}
?>

ejemplo_foreach.php

$colores = array("rojo", "verde", "azul");

Las sentencias break y continue nos permiten controlar el flujo de los bucles.

¿Qué es una Función?

En esta lección, aprenderás todo sobre Funciones en PHP. Las funciones son bloques de código reutilizables que realizan tareas específicas. Nos permiten estructurar mejor el código, evitar la repetición y hacer que el programa sea más claro y modular. Veremos cómo definir y usar funciones, cómo pasar parámetros, cómo retornar valores y cómo funcionan las variables dentro de las funciones.

Definir una función

funcion.php 
<?php
// Definimos una función llamada 'saludar' que no recibe ningún parámetro
function saludar() {
    // Dentro de la función, imprimimos el mensaje "¡Hola, bienvenido a PHP!"
    echo "¡Hola, bienvenido a PHP!";
}
?>

Llamar a una función

llamar_funcion.php 
<?php
// Llamamos a la función 'saludar' que definimos anteriormente
saludar();  // Esto mostrará: ¡Hola, bienvenido a PHP!
?>

Funciones con Parámetros

Las funciones pueden recibir datos (parámetros) para trabajar con ellos dentro de la función. Los parámetros se pasan entre los paréntesis cuando defines y llamas a la función.

Función con un parámetro

funcion_parametro.php 
<?php
// Definimos una función llamada 'saludar' que recibe un parámetro llamado 'nombre'
function saludar($nombre) {
    // Dentro de la función, imprimimos el mensaje "¡Hola, " seguido del nombre recibido, seguido de "!"
    echo "¡Hola, " . $nombre . "!";
}
// Llamamos a la función 'saludar' y le pasamos el nombre "Juan"
saludar("Juan");  // Esto mostrará: ¡Hola, Juan!
?>

Función con varios parámetros

funcion_varios_parametros.php 
<?php
// Definimos una función llamada 'sumar' que recibe dos parámetros, 'a' y 'b'
function sumar($a, $b) {
    // Dentro de la función, imprimimos la suma de 'a' y 'b'
    echo $a + $b;
}
// Llamamos a la función 'sumar' y le pasamos los valores 5 y 3
sumar(5, 3);  // Esto mostrará: 8
?>

Valores por defecto en los parámetros

funcion_valores_defecto.php 
<?php
// Definimos una función llamada 'saludar' que recibe un parámetro llamado 'nombre' con un valor por defecto de "visitante"
function saludar($nombre = "visitante") {
    // Dentro de la función, imprimimos el mensaje "¡Hola, " seguido del nombre recibido, seguido de "!"
    echo "¡Hola, " . $nombre . "!";
}
// Llamamos a la función 'saludar' sin pasarle ningún valor, por lo que usará el valor por defecto
saludar();         // Esto mostrará: ¡Hola, visitante!
// Llamamos a la función 'saludar' y le pasamos el nombre "Desu", por lo que usará este nombre
saludar("Desu");  // Esto mostrará: ¡Hola, Desu!
?>

Funciones con Retorno de Valores

A veces, las funciones necesitan devolver un valor para que puedas trabajar con él. Esto se hace usando la palabra clave `return`.

Función que retorna un valor

funcion_retorno.php 
<?php
// Definimos una función llamada 'multiplicar' que recibe dos parámetros, 'a' y 'b'
function multiplicar($a, $b) {
    // Dentro de la función, devolvemos el resultado de la multiplicación de 'a' y 'b'
    return $a * $b;
}
// Llamamos a la función 'multiplicar' y guardamos el resultado en la variable 'resultado'
$resultado = multiplicar(4, 5);  // Esto devuelve 20
// Imprimimos el resultado
echo $resultado;
?>

Retorno múltiple con `return`

funcion_retorno_multiple.php 
<?php
// Definimos una función llamada 'calcular' que recibe dos parámetros, 'a' y 'b'
function calcular($a, $b) {
    // Dentro de la función, devolvemos un array con la suma de 'a' y 'b' en la primera posición y la multiplicación de 'a' y 'b' en la segunda posición
    return array($a + $b, $a * $b);
}

// Llamamos a la función 'calcular' y guardamos los resultados en las variables 'suma' y 'multiplicacion'
list($suma, $multiplicacion) = calcular(3, 4);  // Esto devolverá 7 y 12
// Imprimimos los resultados
echo "Suma: " . $suma . ", Multiplicación: " . $multiplicacion;
?>

Alcance de las Variables

El **alcance** de una variable se refiere a la parte del programa donde dicha variable es accesible. PHP tiene diferentes tipos de alcance para las variables.

Variables locales

variable_local.php 
<?php
// Definimos una función llamada 'mostrarEdad' que no recibe ningún parámetro
function mostrarEdad() {
    // Dentro de la función, creamos una variable local llamada 'edad' y le asignamos el valor 25
    $edad = 25;  // Variable local
    // Imprimimos el valor de la variable 'edad'
    echo $edad;
}
// Llamamos a la función 'mostrarEdad'
mostrarEdad();  // Esto mostrará: 25
// Intentamos imprimir el valor de la variable 'edad' fuera de la función, lo que causará un error porque 'edad' no existe fuera de la función
echo $edad;     // Esto causará un error porque $edad no existe fuera de la función
?>

Variables globales

variable_global.php 
<?php
// Creamos una variable global llamada 'nombre' y le asignamos el valor "Ana"
$nombre = "Ana";  // Variable global

// Definimos una función llamada 'saludar' que no recibe ningún parámetro
function saludar() {
    // Dentro de la función, usamos la palabra clave 'global' para acceder a la variable global 'nombre'
    global $nombre;
    // Imprimimos un saludo usando el valor de la variable global 'nombre'
    echo "Hola, " . $nombre;
}
// Llamamos a la función 'saludar'
saludar();  // Esto mostrará: Hola, Ana
?>

Variables estáticas (`static`)

variable_estatica.php 
<?php
// Definimos una función llamada 'contador' que no recibe ningún parámetro
function contador() {
    // Dentro de la función, creamos una variable estática llamada 'conteo' y le asignamos el valor 0
    static $conteo = 0;
    // Incrementamos el valor de la variable 'conteo' en 1
    $conteo++;
    // Imprimimos el valor actual de la variable 'conteo'
    echo $conteo;
}

// Llamamos a la función 'contador' tres veces
contador();  // Esto mostrará: 1
contador();  // Esto mostrará: 2
contador();  // Esto mostrará: 3
?>

Práctica: Crear Funciones para Calcular el Área de Figuras Geométricas

Vamos a poner en práctica lo aprendido creando funciones que calculen el área de diferentes figuras geométricas (cuadrado, triángulo y círculo).

Función para el área de un cuadrado

area_cuadrado.php 
<?php
// Definimos una función llamada 'areaCuadrado' que recibe un parámetro llamado 'lado'
function areaCuadrado($lado) {
    // Dentro de la función, devolvemos el área del cuadrado, que es el lado al cuadrado
    return $lado * $lado;
}

// Llamamos a la función 'areaCuadrado' y le pasamos el lado 4
echo "Área del cuadrado: " . areaCuadrado(4);  // Esto mostrará: Área del cuadrado: 16
?>

Función para el área de un triángulo

area_triangulo.php 
<?php
// Definimos una función llamada 'areaTriangulo' que recibe dos parámetros, 'base' y 'altura'
function areaTriangulo($base, $altura) {
    // Dentro de la función, devolvemos el área del triángulo, que es la base por la altura dividido por 2
    return ($base * $altura) / 2;
}

// Llamamos a la función 'areaTriangulo' y le pasamos la base 5 y la altura 8
echo "Área del triángulo: " . areaTriangulo(5, 8);  // Esto mostrará: Área del triángulo: 20
?>

Función para el área de un círculo

area_circulo.php 
<?php
// Definimos una función llamada 'areaCirculo' que recibe un parámetro llamado 'radio'
function areaCirculo($radio) {
    // Dentro de la función, devolvemos el área del círculo, que es pi por el radio al cuadrado
    return M_PI * ($radio * $radio);
}

echo "Área del círculo: " . areaCirculo(3);  // Esto mostrará: Área del círculo: 28.2743
?>

Resumen de la lección

  1. Aprendimos sobre las funciones en PHP y su importancia en la estructuración y reutilización del código.
  2. Estudiamos cómo definir y llamar funciones, cómo pasar parámetros y cómo usar valores por defecto en los parámetros.
  3. Vimos cómo una función puede devolver valores y cómo usar `return` para hacerlo.
  4. Entendimos las diferencias entre variables locales, globales y estáticas, y cómo afectan el alcance de las variables.
  5. Aplicamos lo aprendido creando funciones para calcular el área de diferentes figuras geométricas.

¿Qué es un Array?

Los arrays en PHP son estructuras de datos que permiten almacenar múltiples valores en una sola variable. Esto es especialmente útil cuando necesitas agrupar datos relacionados. En esta lección, aprenderás cómo crear y manejar arrays, recorrer sus elementos y utilizar funciones útiles para trabajar con ellos. También realizaremos una práctica donde implementaremos un sistema de calificaciones utilizando arrays asociativos.

Creación de Arrays

En PHP, existen principalmente dos tipos de arrays: indexados y asociativos.

Arrays Indexados

array_indexado.php 
<?php
// Creamos un array indexado con la función array()
$frutas = array("Manzana", "Banana", "Naranja");
// También puedes usar la sintaxis de corchetes
$frutas = ["Manzana", "Banana", "Naranja"];
?>

Arrays Asociativos

array_asociativo.php 
<?php
// Creamos un array asociativo con la función array()
$calificaciones = array("Juan" => 85, "Ana" => 90, "Luis" => 78);
// También puedes usar la sintaxis de corchetes
$calificaciones = ["Juan" => 85, "Ana" => 90, "Luis" => 78];
?>

Recorrido de Arrays

Para acceder a los elementos de un array, puedes utilizar un bucle. PHP ofrece varias formas de recorrer arrays, como `for`, `foreach`, entre otros.

Usando un bucle `for`

array_for.php 
<?php
// Creamos un array de frutas
$frutas = ["Manzana", "Banana", "Naranja"];

// Usamos un bucle for para recorrer el array
for ($i = 0; $i < count($frutas); $i++) {
    echo $frutas[$i] . "<br>";
}
?>

Usando `foreach`

array_foreach.php 
<?php
// Creamos un array de calificaciones
$calificaciones = ["Juan" => 85, "Ana" => 90, "Luis" => 78];

// Usamos un bucle foreach para recorrer el array
foreach ($calificaciones as $nombre => $calificacion) {
    echo $nombre . ": " . $calificacion . "<br>";
}
?>

Arrays Multidimensionales

Un array multidimensional es un array que contiene uno o más arrays como elementos. Se utilizan para representar estructuras más complejas, como tablas.

Ejemplo de un array multidimensional

array_multidimensional.php 
<?php
// Creamos un array multidimensional de estudiantes
$estudiantes = [
    ["nombre" => "Juan", "calificacion" => 85],
    ["nombre" => "Ana", "calificacion" => 90],
    ["nombre" => "Luis", "calificacion" => 78],
];

// Usamos un bucle foreach para recorrer el array
foreach ($estudiantes as $estudiante) {
    echo $estudiante["nombre"] . ": " . $estudiante["calificacion"] . "<br>";
}
?>

Funciones Útiles para Arrays

PHP ofrece varias funciones que facilitan la manipulación de arrays. Aquí veremos algunas de las más útiles.

`array_push()`

array_push.php 
<?php
// Creamos un array de frutas
$frutas = ["Manzana", "Banana"];
// Usamos la función array_push() para agregar elementos al final del array
array_push($frutas, "Naranja", "Pera");
?>

`array_pop()`

array_pop.php 
<?php
// Usamos la función array_pop() para eliminar el último elemento del array y devolverlo
$ultima_fruta = array_pop($frutas);
echo $ultima_fruta;  // Esto mostrará: Pera
?>

`array_merge()`

array_merge.php 
<?php
// Creamos un segundo array de frutas
$mas_frutas = ["Kiwi", "Mango"];
// Usamos la función array_merge() para combinar los dos arrays
$frutas_totales = array_merge($frutas, $mas_frutas);
?>

`count()`

array_count.php 
<?php
// Usamos la función count() para obtener el número de elementos en el array
$total_frutas = count($frutas);
echo "Total de frutas: " . $total_frutas;  // Muestra el número total de frutas
?>

Práctica: Sistema de Calificaciones

En esta práctica, crearemos un sistema que almacena las calificaciones de los estudiantes en un array asociativo y luego calcularemos el promedio.

Crear el array asociativo

array_calificaciones.php 
<?php
// Creamos un array asociativo con las calificaciones de los estudiantes
$calificaciones = [
    "Juan" => 85,
    "Ana" => 90,
    "Luis" => 78,
    "Marta" => 92,
    "Pedro" => 88,
];
?>

Calcular el promedio

array_promedio.php 
<?php
// Inicializamos la suma y la cantidad de calificaciones
$suma = 0;
$cantidad = count($calificaciones);

// Usamos un bucle foreach para sumar todas las calificaciones
foreach ($calificaciones as $nombre => $calificacion) {
    $suma += $calificacion;  // Sumar todas las calificaciones
}

// Calculamos el promedio
$promedio = $suma / $cantidad;  // Calcular el promedio
echo "Promedio de calificaciones: " . $promedio;  // Mostrar el promedio
?>
