Curso de PHP

¿Qué es PHP?

PHP (Hypertext Preprocessor) es un lenguaje de programación de código abierto, utilizado principalmente para el desarrollo web del lado del servidor. PHP se ejecuta en el servidor, lo que significa que genera HTML u otros tipos de datos (como JSON o XML) que luego se envían al navegador del usuario.

Característica Descripción
Código abierto y gratuito PHP es completamente libre para usar y distribuir.
Fácil de aprender Comparado con otros lenguajes de programación, PHP es fácil de aprender y utilizar para principiantes.
Portabilidad PHP se puede ejecutar en la mayoría de los servidores y sistemas operativos (Windows, Linux, macOS).
Gran soporte comunitario Hay una gran cantidad de recursos disponibles debido a su popularidad.

Historia y evolución de PHP

PHP fue creado en 1994 por Rasmus Lerdorf, inicialmente como un conjunto de scripts CGI (Common Gateway Interface) para rastrear las visitas a su sitio web. Al principio, se llamó "Personal Home Page" (PHP), pero con el tiempo evolucionó a lo que hoy conocemos como Hypertext Preprocessor.

  • PHP 3: Introdujo un motor de scripting que revolucionó su uso.
  • PHP 4: Añadió mejoras de rendimiento significativas.
  • PHP 5: Introdujo características orientadas a objetos.
  • PHP 7: Mejoró drásticamente el rendimiento, reducción del consumo de memoria y nuevas características modernas.

Instalación y configuración de PHP

Para empezar a desarrollar en PHP, necesitamos un servidor web y un entorno donde podamos ejecutar nuestros scripts. Las opciones más comunes son:

  1. XAMPP (Windows, Linux, macOS): Un paquete que incluye Apache (servidor web), MySQL (base de datos), PHP y Perl.
  2. WAMP (Windows): Similar a XAMPP, pero solo para Windows. Incluye Apache, MySQL y PHP.
  3. MAMP (macOS): Paquete de servidor local para macOS, que incluye Apache, MySQL y PHP.

Instalación de XAMPP en Windows:

  1. Descarga XAMPP desde el sitio oficial https://www.apachefriends.org/es/index.html.
  2. Ejecuta el instalador y sigue las instrucciones.
  3. Una vez instalado, abre el Panel de Control de XAMPP y arranca los módulos de Apache y MySQL.
  4. Para probar si PHP está funcionando, crea un archivo llamado index.php en la carpeta htdocs dentro del directorio de XAMPP (por ejemplo, C:\xampp\htdocs\).
  5. Escribe este código dentro de index.php:
index.php 
<?php
    echo "¡Hola, mundo desde PHP!";
?>
  1. Abre tu navegador web y ve a la URL http://localhost/index.php. Si todo está configurado correctamente, verás el mensaje "¡Hola, mundo desde PHP!".

Primer script PHP: Sintaxis básica

PHP utiliza la extensión .php para los archivos. La sintaxis básica de PHP se integra directamente en el HTML usando la etiqueta <?php ... ?>. El código dentro de esta etiqueta se ejecuta en el servidor.

ejemplo.php 
<?php
    // Este es un comentario en PHP
    echo "¡Hola, mundo!";
?>

Explicación del código:

  • <?php ... ?>: Indica el inicio y fin del código PHP dentro de un archivo HTML o PHP.
  • echo: Es un comando en PHP que se utiliza para mostrar texto en la página web.
  • "¡Hola, mundo!": Es una cadena de texto que se muestra cuando se ejecuta el script.

Notas importantes:

  • Los comentarios en PHP pueden ser de una línea (//) o de múltiples líneas (/* ... */).
  • Las instrucciones en PHP suelen terminar con un punto y coma (;).

Integración de PHP con HTML

PHP puede integrarse directamente en un archivo HTML. Esto es lo que hace que PHP sea muy flexible para crear páginas dinámicas.

ejemplo-integrado.php 
<!DOCTYPE html>
<html>
<head>
    <title>Página de prueba en PHP</title>
</head>
<body>
    <h1>Bienvenido a mi página web</h1>
    <p>Este es un párrafo estático en HTML.</p>

    <?php
        echo "<p>Este es un párrafo generado con PHP.</p>";
    ?>
</body>
</html>

Práctica: Escribir un script sencillo

Vamos a crear un pequeño script que combine PHP y HTML:

practica.php 
<!DOCTYPE html>
<html>
<head>
    <title>Mi primera página PHP</title>
</head>
<body>
    <h1>¡Bienvenido a PHP!</h1>

    <?php
        $mensaje = "Este es un mensaje generado dinámicamente con PHP.";
        echo "<p>$mensaje</p>";
    ?>

    <p>¡Espero que disfrutes programando en PHP!</p>
</body>
</html>

Declaración de Variables en PHP

Las variables son espacios en memoria donde almacenamos datos. En PHP, las variables se declaran con el símbolo $. No es necesario declarar el tipo de dato, ya que PHP es de tipado dinámico. Algunas reglas importantes:

  • Deben comenzar con el símbolo $
  • El primer carácter debe ser una letra o guion bajo
  • Son sensibles a mayúsculas y minúsculas
  • No pueden contener espacios, usa guion bajo _ o camelCase
variables.php 
<?php
    $nombre = "Juan";        // String
    $edad = 25;              // Integer
    $precio = 19.99;         // Float
    $es_activo = true;       // Boolean
?>

Tipos de Datos en PHP

PHP soporta varios tipos de datos que puedes utilizar en tus programas. Los más comunes son:

Tipo Descripción Ejemplo
Integer Números enteros $edad = 25;
Float Números decimales $precio = 19.99;
String Cadenas de texto $nombre = "Juan";
Boolean Verdadero o falso $activo = true;
Array Colección de valores $colores = array("rojo", "azul");
NULL Valor especial que representa la ausencia de valor $sin_valor = null;

Manejo de Arrays

PHP tiene dos tipos principales de arrays:

  1. Array indexado: Cada elemento tiene un índice numérico que empieza en 0. Ejemplo: 
    $colores = array("rojo", "azul", "verde");
  2. Array asociativo: Cada elemento tiene una clave (key) en lugar de un índice numérico. Ejemplo: 
    $persona = array("nombre" => "Juan", "edad" => 30, "ciudad" => "Madrid");

Accediendo a los elementos de un array:

<?php
    echo $colores[1]; // Imprime "azul"
    echo $persona['nombre']; // Imprime "Juan"
?>

Operadores en PHP

PHP incluye varios operadores que permiten manipular y comparar valores:

Operadores Aritméticos

Se utilizan para realizar cálculos matemáticos.

Operador Descripción Ejemplo
+ Suma $a + $b
- Resta $a - $b
* Multiplicación $a * $b
/ División $a / $b
% Módulo (resto) $a % $b 
<?php
    $a = 10;
    $b = 3;
    $suma = $a + $b;  // Resultado: 13
    $resta = $a - $b; // Resultado: 7
    $multiplicacion = $a * $b; // Resultado: 30
    $division = $a / $b; // Resultado: 3.333...
    $modulo = $a % $b; // Resultado: 1
?>

Operadores de Asignación

Sirven para asignar valores a las variables.

Operador Descripción Ejemplo
= Asignación simple $x = $y
+= Suma y asigna $x += $y (equivale a $x = $x + $y)
-= Resta y asigna $x -= $y
*= Multiplica y asigna $x *= $y
/= Divide y asigna $x /= $y 
<?php
    $x = 5;
    $x += 2; // $x ahora es 7
    $x *= 3; // $x ahora es 21
?>

Operadores de Comparación

Se utilizan para comparar valores. El resultado siempre será true o false.

Operador Descripción Ejemplo
== Igual a $a == $b
!= Diferente de $a != $b
> Mayor que $a > $b
< Menor que $a < $b
>= Mayor o igual que $a >= $b
<= Menor o igual que $a <= $b 
<?php
    $a = 10;
    $b = 5;
    $resultado = ($a == $b); // false
    $resultado = ($a != $b); // true
    $resultado = ($a > $b); // true
    $resultado = ($a < $b); // false
    $resultado = ($a >= $b); // true
    $resultado = ($a <= $b); // false
?>

Operadores Lógicos

Se usan para combinar múltiples condiciones lógicas.

$condicion1 = true;
$condicion2 = false;

$resultado = $condicion1 && $condicion2; // AND: false
$resultado = $condicion1 || $condicion2; // OR: true
$resultado = !$condicion1;               // NOT: false

Conversión de Tipos en PHP

PHP puede convertir tipos de datos automáticamente cuando sea necesario (conversión implícita), pero también puedes hacerlo de forma manual:

$num_str = "10";    // Cadena
$numero = $num_str + 5; // PHP convierte "10" a número automáticamente
echo $numero;       // Resultado: 15

// Conversión explícita
$num_str = "3.14";
$flotante = (float)$num_str;  // Convertimos a flotante
echo $flotante;  // Resultado: 3.14

?>

<?php
$numeros = array(5, 10, 15);

    echo "La primera condición es verdadera.";
} else {
    echo "La condición no se cumple.";
}
?>
